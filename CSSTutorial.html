Tutorial CSS

1. What is CSS

CSS sau Cascading Style Sheets il folosim pentru a stiliza
pagina noastra web, practic ii spunem browserului cum sa prezinte utilizatorului documentul html. Deci, folosim CSS
pentru a stiliza textul si alte elemente din documentul html, pentru a randa continutul, pentru a colora pagina
web si asa mai departe.

2. CSS Inclusion (in HTML documents)

Metoda cea mai recomandata este sa folosim un fisier de still extern pentru a include CSS in documentul HTML. Pentru aceasta, trebuie sa folosim elemntul <link> in cadrul elementului <head>

3. CSS Syntax (Selector, property, value)

Sintaxa CSS arata asa:

Selector - folosim selectori pentru a selecta anumite elemente html aplicandule astfel un anumit set de reguli.

property - este proprietatea selectorului sau cum dorim sa il stilizam.
Pentru o lista lunga cu CSS property putem cauta documentatia pe google :)

value - este valoare specifica fiecarei proprietati.

4. CSS Selectors (type, universal, descendant, id, class, child, attribute)

Folosim selectori pentru a selecta anumite elemente pentru a le stiliza.

Putem imparti CSS Selectors in 6 categorii:

1. Universal Selector
Selecteaza toate elementele
2. Simple selector
selecteaza elemente in functie de element HTml, id si clasa.
3. Combinator selectors
selecteaza elemente pe baza unei relatii specifice intre ele.
4. Pseudo-class selectors
selecteaza elementele care se afla intr-o anumita stare.
5. Pseudo-elements selectors
selecteaza si aplica stil unei parti a elementului.
6. Attribute selectors
selecteaza elemente pe baza unui atribut sau a valorii unui atribut.

5. CSS Measurement units (inches, pixels, points, rem, em, percent, vw, vh, vmin, vmax )

O unitate CSS determina dimensiunea unui element sau a continutului acestiua in browserul nostru.
Exista doua tipuri de unitati: absolute si relative.

  1. Unitatile absolute: sunt unitati cu o masura fixa de aceea sunt mai putin folosite deoarece acestea
    nu scaleaza atunci cand dimensiunea ectranului se modifica.
    • px(pixels): mai sunt si alte unitati absolute dar "px" este cea mai folosita unitate absoluta pentru ecrane.
    • in(inches): 1in = 96px (depinde de densitatea ecranelor)
    • pt(points): 1in = 72pt
  2. Unitatile relative: specifica o lungime relativa la o alta proprietate de lungime. Unitatile relative sunt utile
    pentru modelarea site-urilor receptive, deoarece se scaleaza in raport cu dimensiunea parinte sau fereastra.
    • rem: unitatea rem este relativa la font-size-ul elementului radacina adica <html>
    • em: unitatea em este relativa la font-size-ul in care este elementul (adica 2em este de 2 ori mai mare decat font-ul curent)
    • % (percent): unitatea % este relativ la elementul parinte
    • vw: este relativ cu 1% din latimea ferestrei de vizualizare
    • vh: este relativ cu 1% din inaltimea ferestrei de vizualizare
    • vmin: este relativ cu 1% din dimensiunea mai mica a ferestrei de vizualizare
    • vmax: este relativ cu 1% din dimensiunea mai mare a ferestrei de vizualizare

6. CSS Colors and Backgrounds

CSS Colors

La fel cum am prezentat si in tutorialul de HTML, culorile pot fi specificate cu nume de culori sau cu valori
RGB, RGBA, HEX, HSL si HSLA, dar de data acestea le puteam specifica ca si set de reguli din fisierul css.

CSS Backgrounds

Proprietatile CSS Backgrounds sunt folosite pentru a adauga efecte de fundal pentru elemente.

CSS Background are urmatoarele proprietati:

  • background-color:
    seteaza culoarea de fundal a unui element.
  • background-image:
    seteaza o imagine ca fundal al unui element.
  • background-origin:
    specifica pozitia de origine (zona de pozitionare a fundalului) a unei imagini de fundal.
  • background-position:
    setează pozitia initială pentru fiecare imagine de fundal. Pozitia este relativa la stratul de pozitie stabilit de background-origin.
  • background-repeat:
    repeta o imagine atat pe orizontala, cat si pe verticala.
  • background-size:
    specifica dimensiunea imaginilor de fundal.
  • background-clip:
    ne permite sa controlam cat de mult se extinde o culoare/imagine de fundal dincolo de continutul unui element.
  • background-attachment:
    stabileste daca pozitia unei imagini de fundal este fixa in fereastra de vizualizare sau deruleaza cu blocul care o contine.
  • backround-blend-mode:
    defineste modul in care imaginea de fundal a unui element ar trebui sa se imbine cu culoarea de fundal:
  • background Shorthand:
    proprietatea shorthand ne permite sa scurtam codul prin a specifica toate proprietatile de fundal la un loc.
    Cand folosim proprietatea shorthand ordinea valorilor proprietatii este:
    1. background-color
    2. background-image
    3. background-position
    4. background-size
    5. background-repeat
    6. background-attachment
    7. background-origin
    8. background-clip
    9. initial: seteaza aceasta proprietate la valoarea implicita. / inherit: mosteneste proprietate din elementul parinte.

7. CSS Text and Fonts

CSS Text

Stilizarea textului in CSS este ceva pe care il vom face destul de des, deoarece majoritatea continutului de pe o
pagina web este in text.
Din fericire, CSS are o multime de proprietati pentru formatarea textului.

Ne vom uita la urmatoarele proprietati pentru formatarea textului:

  • text color
    - este folosit pentru a seta culoarea unui text.
  • text-align
    - este folosit pentru a alinia textul unui document.
  • text-align-last
    - este folosit pentru a specifica modul in care se aliniaza ultima linie a unui text.
  • text-decoration
    - este folosit pentru a specifica decorul adaugat textului: underline, overline, strikethrough
  • text-transform
    este folosit pentru a specifica litere mari si mici intr-un text.
  • text-indent
    - este folosit pentru a indenta prima linie dintr-un bloc de text.
  • text-justify
    - este folosit pentru a specifica metoda de justificare a textului cand text-align este setat la „justify”.
  • text-orientation
    - seteaza orientarea caracterelor textului intr-o linie.
    Afecteaza doar textul in modul vertical "writing-mode: vertical-rl;".
  • text-shadow
    este folosit pentru a adauga umbra textului.
  • text-overflow
    este folosit pentru a specifica modul in care continutul depasit care nu este afisat ar trebui sa fie semnalat utilizatorului.
  • text-transform
    este folosit pentru a specifica litere mari si mici într-un text.
  • line-hight
    este folosit pentru a specifica inaltimea unei linii.
  • letter-spacing
    este folosit pentru a adauga sau scadea spatiu intre literele care alcatuiesc un cuvant.
  • word-spacing
    este folosit pentru a adauga sau scadea spatiu intre cuvintele unei propozitii.
  • direction
    este folosit pentru a seta directia textului.

CSS Fonts

Este important sa folosim un font usor de citit si unul care sa creeze un identitate puternica contentului

Ne vom uita la urmatoarele proprietati ale fontului:

  • font-family
    este folosit pentru a specifica fontul unui text.
    Proprietatea font-family ar trebui sa detina mai multe nume de fonturi ca
    si backup pentru a asigura compatibilitatea intre browsere sau sisteme de operare.
  • font-size
    este folosit pentru a seta dimensiunea textului.
  • font-weight
    este utilizat pentru a seta cat de groas sau subtire ar trebui sa fie afisate caracterele din text.
  • font-style
    este folosit pentru a specifica stilul fontului pentru un text.

O modalitate foarte convenabila si usoara de a adauga fonturi web pe site-ul nostru sunt
Google Fonts este principala gazda de fonturi open-source pentru utilizare pe web cu sute de
font-family disponibile pentru a include in pagina noastra web.

9. CSS Links

Folosind CSS, Link-urile pot fi stilizate in mai multe moduri diferite.

  • link text implicit
  • link text stilizat
  • link buton implicit
  • link buton stilizat

Linkurile pot fi stilizate in mode diferit, in functie de starea in care se afla.

Cele patru stari sunt:

  1. a:link - un link normal, nevizitat
  2. a:visited - un link pe care utilizatorul l-a vizitat
  3. a:hover - un link atunci cand user-ul trece mouse-ul peste el
  4. a:active - un link in momentul in care se face clic pe acesta

16. CSS Positioning

Proprietatea position specifica tipul de metoda de pozitionare utilizata pentru un element bloc.

Exista cinci valori diferite de pozitie:

  1. static
    aceasta positie vine in mod implicit si este intotdeauna pozitionata conform fluxului normal al paginii web.
  2. relative
    un element cu poziție "relative" este pozitionat fata de pozitia sa normala prin setarea proprietatilor: top, right, bottom, left
  3. absolute
    un element cu poziție "absolute" este pozitionat relativ la cel mai apropiat parinte cu pozitie "relative"
  4. fixed
    este pozitionat relative fata de fereastra web(viewport), astfel va ramane intotdeauna in acelasi loc chiar daca pagina este glisata .
  5. sticky
    acesta se comporta ca si poszitia "fixed" doar ca este pozitionat pe baza pozitiei de glisare a utilizatorului.

exemplu CSS Positioning

19. CSS @Rules

CSS @Rules sunt foarte practice in ceea ce priveste spunerea CSS-ului cum sa se comporte.
Exista mai multe dintre aceste reguli, cum ar fi:

  • @charset
    Regula @charset trebuie sa fie primul element din foaia de stil si sa nu fie precedata de niciun caracter.
    Daca sunt definite mai multe reguli @charset, este folosita doar prima.
    Acest lucru este util in cazul in care encoding-ul nu este declarat in meta tag-ul header-ului.
  • @document
    Regula @document este folosita pentru a specifica un set de stiluri care vor fi aplicate numai unor anumite
    pagini ale unui site web care au o adresa URL pe care o specificam.
  • @font-face
    Regula @font-face este folosita pentru a asocia un nume de font care poate fi folosit intr-o foaie de stil.
    Acesta poate fi folosita cu fonturi descarcate local iar in file CSS vom utiliza o descriere
    font-family:'' si un src: url(/locatia fonturilor descarcate).
  • @import
    Regula @import este folosita pentru a importa reguli de stil astfel avem:
    • @import reguli css dintr-un alt fisier css
    • @import regli css dintr-un URL
    • @import media queries dependent de fisierul css curent
  • @keyframes
    determina ce stiluri va avea un element la un moment dat. @keyframes sunt responsabile pentru controlul etapelor
    intermediare ale animatiei, de asemenea, permite un control mai mare al progresului unei animatii.
  • @media
    regula @media este folosiota pentru a ajusta designul web pentru a se potrivi cu dispozitivele utilizatorilor
  • @namespace
    poate fi folosit pentru a defini spatiul de nume implicit pentru foaia de stil. Cand este definit un spatiu de nume
    implicit, toti selectorii de tip si universali se aplica numai elementelor din acel spatiu de nume.
  • @page
    este folosita pentru a modifica unele proprietati CSS la printarea unui pagini web.
  • @supports
    este folosit pentru a testa disponibilitatea anumitor caracteristici CSS in interiorul unui browser in CSS,
    adica facem detectarea caracteristicilor in cadrul css-ului

exemplu CSS @Rules

20. CSS Media Types

Ne permit sa decidem cum ar trebui sa arate o pagina, in functie de tipul de media utilizat pentru afisarea paginii

  • @media all
    in cele mai multe cazuri, @media all este utilizat in mod implicit @media atunci cand nu este specificat
    niciun alt tip sau opeator.
  • @media screen
    destinat in principal pentru ecran PC, leptop, tableta, smart-phones ...
  • @media print
    folosit pentru documentele care sunt vizualizate pe un ecran in modul de previzualizare la imprimare.
    Exemplu, daca vrem sa economisim culoarea la printare putem avea regula de mai jos:
  • @media type aural braille handheld
    Note: CSS2.1 si Media Queries 3 au definit mai multe tipuri de media suplimentare
    (tty, tv, projection, handheld, braille, embossed, and aural), dar au fost depreciate
    in Media Queries 4 si nu ar trebui sa fie folosite.
