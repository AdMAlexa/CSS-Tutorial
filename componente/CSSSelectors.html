CSS Selectors

1. Universal selector

Selecteaza toate elementele

2. element Selectors

Element Selector selecteaza elementul sau elementele de acelasi tip html de pe pagina web pentru a le stiliza.

Putem avea group selectors pentru a aplicat acelasi stil la mai multe elemente html.

4. id selector

Selectorul de tip id foloseste atributul id al unui element HTML pentru a selecta un anumit element.
Id-ul unui element este unic intr-o pagina.

3. class selector

Selectorul de tip clasa selecteaza elementele in functie de continutul atributului lor de clasa.

Putem avea mai multe clase pe un element html.

4. Combinators CSS

Un selector CSS poate contine mai mult de un selector simplu printre care putem include un combinator.

Avem patru combinatoare diferite in CSS:

  1. descendant selector: selecteaza toate elementele/copii, nepotii si stranepotii al elementului parinte specific
  2. child selector: selecteaza doar copii directi ai elementului parinte specific

    vedem rezultatul html la exemplu 2
  3. adjacent sibling selector: este folosit pentru a selecta un element care se afla direct dupa un alt element specific.

  4. general sibling selector: selecteaza toate elementele care sunt urmatorii frati ai unui element specificat.

5. Pseudo-class selectors

Pseudo-class selectors sunt folositi pentru a defini o stare speciala a unui element.

:first-child

Aceasta selecteaza primul copil al unui alt element.

:last-child

Aceasta selecteaza ultimul copil al unui alt element.

:nth-child(n)

Aceasta selecteaza al n-lea copil al părintelui său.
n poate fi un nr., cuvânt cheie sau o formula.

:hover

- este folosit pentru a selecta elemente atunci cand glisam cu mouse-ul peste ele.

:focus

- este declansat atunci cand utilizatorul face clic, atinge sau il selecteaza de la tastataura.

6. Pseudo-elements selectors

Pseudo-elements selectors sunt folositi pentru a aplica un anumit stil unei parti specifice a unui element.

::before

- folosit pentru a introduce un anumit continut inaintea continutului unui element.

::after

- folosit pentru a introduce un anumit continut dupa continutul unui element.

::first-line

- folosit pentru a adauga un stil la prima linie a unui text.

::selection

- folosim pentru marcarea unei portiuni a unui element care este selectata de un utilizator.
